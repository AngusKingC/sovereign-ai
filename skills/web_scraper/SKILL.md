# Web Scraper Skill

## Description
Scrapes webpage content using HTTP requests and HTML parsing with BeautifulSoup.

## Parameters
- url (str, required): The URL to scrape
- selector (str, optional): CSS selector for targeted scraping

## Output
Returns scraped text content as a string.

## Dependencies
- httpx>=0.24.0
- beautifulsoup4>=4.12.0

## Hardware
No special hardware requirements.
