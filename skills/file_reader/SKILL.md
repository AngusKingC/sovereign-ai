# File Reader Skill

## Description
Reads local file contents with configurable encoding.

## Parameters
- path (str, required): Path to the file to read
- encoding (str, optional): File encoding, defaults to utf-8

## Output
Returns file content as a string.

## Dependencies
None (uses built-in file I/O)

## Hardware
No special hardware requirements.

## Tags
- file-io
- data-reading
- local
