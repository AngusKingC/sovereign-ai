# File Writer Skill

## Description
Writes content to local files with configurable mode.

## Parameters
- path (str, required): Path to the file to write
- content (str, required): Content to write to the file
- mode (str, optional): File write mode, defaults to 'write'

## Output
Returns a tuple of (success: bool, bytes_written: int).

## Dependencies
None (uses built-in file I/O)

## Hardware
No special hardware requirements.
